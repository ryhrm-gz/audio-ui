# EQCurve

`EQCurve` is a headless editor for parametric EQ bands. It maps frequency on a logarithmic X axis, gain on a vertical dB axis, and generates deterministic preview curve points for bell, shelf, and pass-filter bands.

<EQCurveDemo />

## Anatomy

```tsx
import { EQCurve } from "@ryhrm-gz/audio-ui-react";

const bands = [
  { id: "low", type: "low-shelf", frequency: 120, gain: 3, q: 0.8 },
  { id: "mid", type: "bell", frequency: 900, gain: -4, q: 1.4 },
  { id: "air", type: "high-shelf", frequency: 9000, gain: 4, q: 0.7 },
];

export function ChannelEQ() {
  return (
    <EQCurve.Root defaultValue={bands} name="channel-eq">
      <EQCurve.Graph>
        <EQCurve.Grid />
        <EQCurve.Curve />
        <EQCurve.Bands>{(band) => <EQCurve.Band key={band.id} band={band} />}</EQCurve.Bands>
      </EQCurve.Graph>
      <EQCurve.Value />
      <EQCurve.HiddenInput />
    </EQCurve.Root>
  );
}
```

## Band model

Each band is normalized by the core package before it reaches render state. Missing `enabled` values are treated as `true`.

```ts
type EQCurveBandType = "bell" | "low-shelf" | "high-shelf" | "low-pass" | "high-pass";

type EQCurveBand = {
  id: string;
  type: EQCurveBandType;
  frequency: number;
  gain: number;
  q: number;
  enabled?: boolean;
};
```

## Keyboard interaction

Focus a band and use Arrow Left/Right to change frequency and Arrow Up/Down to change gain. Page Up and Page Down adjust Q by a larger step. Hold Shift for fine control. Hold Alt with arrow keys to adjust Q directly.

## Styling hooks

The curve is rendered as an SVG path by default, and each band exposes normalized position variables from `0` to `1`.

```css
.eq-band {
  left: calc(var(--eq-curve-band-x) * 100%);
  top: calc((1 - var(--eq-curve-band-y)) * 100%);
}

.eq-curve [data-part="curve-path"] {
  stroke: currentColor;
}
```

## API reference

### `EQCurve.Root`

`EQCurve.Root` renders a `div`, owns band state, and provides context to the other parts. It accepts standard `div` props except `defaultValue`, `onChange`, and `children`, which are replaced by the EQ-specific API.

| Prop              | Type                                | Default | Description                                                                                            |
| ----------------- | ----------------------------------- | ------- | ------------------------------------------------------------------------------------------------------ |
| `value`           | `EQCurveBand[]`                     | -       | Controlled band array. Pass with `onValueChange` when external state owns the EQ bands.                |
| `defaultValue`    | `EQCurveBand[]`                     | `[]`    | Initial uncontrolled band array.                                                                       |
| `minFrequency`    | `number`                            | `20`    | Lower frequency bound in hertz. Frequency positions use a logarithmic scale.                           |
| `maxFrequency`    | `number`                            | `20000` | Upper frequency bound in hertz.                                                                        |
| `minGain`         | `number`                            | `-24`   | Lower gain bound in dB.                                                                                |
| `maxGain`         | `number`                            | `24`    | Upper gain bound in dB.                                                                                |
| `minQ`            | `number`                            | `0.1`   | Lower Q bound.                                                                                         |
| `maxQ`            | `number`                            | `18`    | Upper Q bound.                                                                                         |
| `stepFrequency`   | `number`                            | `1`     | Frequency increment for pointer and keyboard changes.                                                  |
| `stepGain`        | `number`                            | `0.1`   | Gain increment for pointer and keyboard changes.                                                       |
| `stepQ`           | `number`                            | `0.1`   | Q increment for keyboard changes.                                                                      |
| `curveResolution` | `number`                            | `128`   | Number of preview points generated for `EQCurve.Curve`.                                                |
| `disabled`        | `boolean`                           | `false` | Prevents pointer and keyboard interaction, removes bands from tab order, and adds disabled attributes. |
| `readOnly`        | `boolean`                           | `false` | Keeps bands focusable while preventing value changes.                                                  |
| `fineControl`     | `boolean`                           | `true`  | Enables reduced step size while Shift is held during drag or keyboard interaction.                     |
| `name`            | `string`                            | -       | Default form field name consumed by `EQCurve.HiddenInput`.                                             |
| `required`        | `boolean`                           | -       | Default required flag consumed by `EQCurve.HiddenInput`.                                               |
| `children`        | `ReactNode \| (state) => ReactNode` | -       | Parts to render. Function children receive the render state, including bands, curve points, and flags. |
| `onValueChange`   | `(value: EQCurveBand[]) => void`    | -       | Fires when interaction produces a normalized band array.                                               |
| `onValueCommit`   | `(value: EQCurveBand[]) => void`    | -       | Fires when an interaction commits, such as pointer up or keyboard step.                                |
| `render`          | `RenderProp`                        | -       | Replaces the root element while preserving generated props and render state.                           |

### Parts

| Part                  | Props and behavior                                                                                                                                           |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `EQCurve.Graph`       | Renders a `div` editing area. Band dragging reads this element's size so rendering can be responsive.                                                        |
| `EQCurve.Grid`        | Renders an `aria-hidden` `div` for grid styling. It does not generate lines itself; use CSS backgrounds, pseudo-elements, or custom `render`.                |
| `EQCurve.Curve`       | Renders an `svg` with a generated path by default. Pass children or `render` to own the curve drawing while using `state.curve` points.                      |
| `EQCurve.Bands`       | Renders one `EQCurve.Band` per band by default. Function children receive each normalized band object for custom rendering.                                  |
| `EQCurve.Band`        | Renders a focusable `span` with `role="slider"`, generated ARIA metadata, keyboard handling, pointer dragging, band data attributes, and position variables. |
| `EQCurve.Value`       | Renders a `span` connected to bands by `aria-describedby`. `format(value, state)` customizes display text when explicit children are not supplied.           |
| `EQCurve.HiddenInput` | Renders `<input type="hidden">` with the normalized band array serialized as JSON. `name`, `required`, and `disabled` default to the matching root props.    |

### CSS variables

| Variable                    | Component       | Description                               |
| --------------------------- | --------------- | ----------------------------------------- |
| `--eq-curve-band-count`     | `Root`          | Number of normalized bands                |
| `--eq-curve-min-frequency`  | `Root`, `Graph` | Lower frequency bound                     |
| `--eq-curve-max-frequency`  | `Root`, `Graph` | Upper frequency bound                     |
| `--eq-curve-min-gain`       | `Root`, `Graph` | Lower gain bound                          |
| `--eq-curve-max-gain`       | `Root`, `Graph` | Upper gain bound                          |
| `--eq-curve-band-frequency` | `Band`          | Band frequency in hertz                   |
| `--eq-curve-band-gain`      | `Band`          | Band gain in dB                           |
| `--eq-curve-band-q`         | `Band`          | Band Q value                              |
| `--eq-curve-band-x`         | `Band`          | Normalized logarithmic frequency position |
| `--eq-curve-band-y`         | `Band`          | Normalized gain position from `0` to `1`  |
